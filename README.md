# Guess Protocol 🎯

Guess Protocol is a browser-based number guessing race where you compete against an adaptive AI. It gives players a quick, fair, and satisfying way to test their deduction skills while the AI tries to uncover their secret number first.

🔗 [Play Guess Protocol](./index.html)

## Features

* ✨ **Human vs AI race** – You and the AI take one turn each round to find the other player's secret number.
* 🔒 **Verified secret number** – Your number is locked in the browser, so higher, lower, and correct feedback always stays fair.
* 🧠 **Four AI difficulty modes** – Play against random, binary-search, human-prediction, or hybrid AI strategies.
* 📊 **Live game intelligence** – Follow your known range, current round, and how much of the AI's search space has been eliminated.
* 📱 **Responsive game board** – Play comfortably on desktop or mobile.
* 🎉 **Satisfying interactions** – Enjoy animated buttons, confetti wins, and a clear end-of-game result screen.

## Run locally

Open `index.html` in a modern web browser. No installation or external libraries are required.
